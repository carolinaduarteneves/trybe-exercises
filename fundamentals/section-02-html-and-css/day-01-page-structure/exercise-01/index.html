My HTML Learning Page

Hello everyone!
This page was created as a practice project to train my HTML skills and understand the basic structure of a web page. The goal of this website is to practice how content is organized using headings, paragraphs, lists, links, and images. The concepts applied here were learned during my studies at Trybe.

HTML Elements Practiced

  • Headings (h1, h2)
  • Paragraphs
  • Internal and external links
  • Ordered and unordered lists
  • Images
  • Line breaks

Steps to Create This Page

  1. Create the HTML file
  2. Add the basic page structure
  3. Insert headings and paragraphs
  4. Add lists, links, and images
  5. Review and organize the content

Why HTML Is Important

  • Defines the structure of a web page
  • Organizes content clearly
  • Works together with CSS and JavaScript
  • Is the foundation of web development
