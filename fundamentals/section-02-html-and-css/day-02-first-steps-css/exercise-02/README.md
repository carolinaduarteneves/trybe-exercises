## Overview
On this day, I focused on learning the basics of CSS and how it is used to style HTML pages.
The goal was to understand how CSS controls the visual appearance of a web page, including colors, fonts, spacing, and layout.

## What I learned
- How to apply CSS styles to HTML elements
- How to use basic CSS properties such as color, background-color, and font-size
- How to style text using fonts and text alignment
- How to use classes and selectors to apply styles
- The difference between inline, internal, and external CSS
- How CSS improves the visual organization of a page

## Practice
- Styled a basic HTML page using CSS
- Applied colors, fonts, and spacing to improve readability
- Practiced using CSS selectors and classes
- Experimented with visual customization of page elements

## Key takeaway
CSS allows me to transform a simple HTML structure into a visually organized and pleasant web page. Understanding CSS basics is essential to creating attractive and user-friendly websites.
